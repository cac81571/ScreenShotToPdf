# マイグレーション スクリーンショット → PDF

システムマイグレーションのスクリーンショットを、**移行前**と**移行後**で比較しやすい1本のPDFにまとめるツールです。

- **奇数ページ**: 移行前フォルダの画像（ファイル名でソート）
- **偶数ページ**: 移行後フォルダの画像（ファイル名でソート）
- 対応画像: PNG, JPG, JPEG, GIF, BMP, WebP

## 技術要素

- **Groovy** (4.x) + **Swing**（GUI）
- **Maven**（ビルド）
- **OpenPDF**（PDF生成）

## ビルド・実行

```bash
# ビルド（依存関係をすべて含む起動可能 JAR が target に生成される）
mvn package -DskipTests

# 実行（単体 JAR で起動）
java -jar target/migration-screenshot-pdf-1.0.0-SNAPSHOT.jar
```

開発時は `mvn exec:java` でも起動できます。

## 使い方

1. 起動すると Swing のウィンドウが開きます。
2. **移行前フォルダ** … 移行前のスクリーンショットが入ったフォルダを「参照...」で指定。
3. **移行後フォルダ** … 移行後のスクリーンショットが入ったフォルダを「参照...」で指定。
4. **出力PDF** … 作成する PDF の保存先ファイルを指定（未指定時は任意の名前で保存ダイアログから選べます）。
5. **PDF作成** ボタンを押すと、上記のルールで PDF が生成されます。

同一インデックス（ファイル名ソート順）の画像が、奇数ページ（移行前）と偶数ページ（移行後）で並ぶため、見比べながら確認しやすくなります。
